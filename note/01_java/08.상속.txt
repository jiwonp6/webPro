◎ 상속(extends)
	(overloading vs. overriding)
1. 상속
	: 다른 클래스로 부터 해당 클래스로 속성과 메소드 상속하여 재사용, 
	  여러 객체를 하나의 객체로 묶을 수 있음(추상화)
	* 문법
	접근제어자 [final/abstract] class 클래스명 extends 상위클래스명{
		추가할 멤버변수 선언;
		생성자;
		추가할 메소드 선언;
	}
2. 오버라이드(Override : 재정의)
	: 부모 클래스의 메소드 → 자식 클래스에서 (동일이름으로) 재정의, 
				부모 클래스의 메소드 사용은X
3. 오버로드(Overload : 중복정의)
	: 인자 타입 다를 때 → 메소드 같은이름, 다른 기능으로 중복정의O
4. super
	: 부모를 찾는 키워드
	* 사용
	override에 의해 부모 클래스이 메소드를 사용하지 못할 때 사용
