◎ 접근제한자와 static
1. 접근제한자
	: 클래스의 데이터, 메소드에 대하여 다른 클래스로부터의 접근을 조정
	* 종류
	- public
		: 제한X
	- default
		: 해당 클래스와 동일한 패키지나 상속받은 클래스의 경우에만 public 권한 처럼 사용O
	- private
		: 해당 클래스만 사용O, 외부 객체 접근X
2. static
	: 클래스 변수, 한 클래스 안의 모든 객체는 클래스 변수 공유, 클래스 이름을 통해 접근
	but, Garbege Collector의 관리X → 항상 메모리에 상주
	* 사용
		: 객체들끼리 공유하고 싶은 정보가 있을 때 사용
		  즉, 새롭게 객체 생성X
