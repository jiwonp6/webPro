◎ 함수
	: 코드의 집합 나타내는 자료형
1. 익명 함수
	: var functionVar = function() { };
		* 이름X → 변수에 넣어 사용
2. 선언적 함수
	: function 함수명 ( ) { }
3. 매개변수와 리턴값
	: function 함수이름(매개변수, 매개변수, 매개변수) {
		[함수 코드]
		return 리턴값;
	}
		* 리턴값X → 익명함수 이용
4. 매개변수
	: 원 함수의 매개변수보다 ↑ → 매개변수 무시
	  원 함수의 매개변수보다 ↓ → undefined
		* Array()함수 ; 빈 배열을 만든다.
		  Array(number) ; 매개변수만큼의 크기를 가지는 배열을 만든다.
		  Array(any, …, any); 매개변수를 배열로 만든다.
5. 가변인자 함수
	: 매개변수의 개수 변경 O	eg. Array()함수
		* arguments : 매개변수 배열
6. 리턴값
	: return;
7. 내부 함수
	: 함수 내부에 또 함수 선언, 내부함수가 우선시 됨
		* jQuery는 대부분 선언적 함수를 내부함수로 작성하고 있음
		  (jQuery = MIT 라이선스로 배포되는 무료 오픈소스 소프트웨어)
		* 자기호출함수
			: (fuction(){ … })(); 
8. 콜백 함수
	: 함수의 매개변수로 함수가 사용되는 것, 익명함수로 많이 사용됨
	
	※ 내부함수 	v.s. 	콜백함수
	   function() {		function_name( function(){ } );
	        function(){ }	
	   }
